# Mapjiri 해커톤 발표 자료

---

## 1. 서비스 소개

### 서비스명
**Mapjiri (맵지리)**

### 한 줄 소개
> 경인지역 대학생들의 캠퍼스 성장을 지도처럼 안내하는 올인원 플랫폼

### 배경 / 문제 정의
- 대학생들은 공모전, 스터디, 세미나, 대관 등 정보가 **분산**되어 있어 찾기 어려움
- 선배/멘토와의 연결 창구가 없어 진로 고민을 혼자 해결해야 함
- 활동 참여 이력이 체계적으로 쌓이지 않아 포트폴리오 관리가 어려움

### 해결책
- 공모전·세미나, 스터디·동아리 모집, 공간 대관, 선배 상담을 **한 곳에서** 제공
- 활동할수록 **코인과 레벨**이 쌓이는 성장 시스템으로 동기 부여
- **AI 추천** 기능으로 개인 맞춤 활동 제안

---

## 2. 주요 기능

### 🏠 홈화면 (대시보드)
- 사용자 레벨 및 코인 현황 표시 (뚜벅이 → 씽씽이 → 따릉이 → 봉봉이 → 우주대장)
- 근처 활동 지도 (경인지역 기반)
- 최근 활동 피드

### 🏆 공모전 · 세미나
- AI가 관심 분야 기반으로 공모전 자동 추천
- D-Day 카운트다운, 카테고리 필터, 검색
- 세미나 참여 시 코인 획득
- 북마크(찜) 기능

### 👥 모집 (스터디 · 동아리)
- 스터디/동아리/프로젝트 팀원 모집 게시판
- 태그 기반 필터링
- AI 홍보 게시글 자동 생성

### 🏢 대관 서비스
- 경인지역 세미나실·스터디룸·회의실 예약
- 레벨별 할인 적용 (뚜벅이 5% ~ 우주대장 25%)
- 실시간 예약 현황 확인

### 💬 선배 상담
- 취업·진로·스펙 관련 고민 작성
- AI 기반 멘토 매칭
- 채팅형 상담 인터페이스
- 상담 완료 시 코인 지급

### 📊 마이페이지
- 성장 레벨 시각화 (5단계)
- 코인 적립 내역
- 참여한 활동 이력 관리

## 3. 기술 스택

### Frontend
| 항목 | 기술 |
|---|---|
| 프레임워크 | React 18 + TypeScript |
| 빌드 도구 | Vite |
| 스타일링 | Tailwind CSS |
| UI 컴포넌트 | shadcn/ui (Radix UI 기반) |
| 라우팅 | React Router v7 |
| 상태관리 | Zustand |

### Backend
| 항목 | 기술 |
|---|---|
| 런타임 | Node.js |
| 프레임워크 | Express.js |
| 언어 | TypeScript |
| AI | Google Gemini 1.5 Flash API |

### Database
| 항목 | 기술 |
|---|---|
| DB | MySQL 8.0 |
| ORM | mysql2/promise (직접 쿼리) |

### 인프라 / 배포
| 항목 | 기술 |
|---|---|
| 컨테이너 | Docker + Docker Compose |
| 웹 서버 | nginx (리버스 프록시) |
| 클라우드 | AWS EC2 (t3.small) |
| 리전 | 서울 (ap-northeast-2) |

## 5. AI 기능

## 6. 성장 시스템 (레벨 · 코인)

| 레벨 | 이름 | 필요 코인 | 대관 할인 |
|---|---|---|---|
| Lv.1 | 뚜벅이 (걷다) | 0 | 5% |
| Lv.2 | 씽씽이 (킥보드) | 50 | 10% |
| Lv.3 | 따릉이 (자전거) | 200 | 15% |
| Lv.4 | 봉봉이 (자동차) | 500 | 20% |
| Lv.5 | 우주대장 (우주선) | 1,000 | 25% |

**코인 획득 방법**
- 선배와 상담 완료: +30 코인
- 세미나 시청/참석: +15~30 코인
- 동아리 활동 인증: +15 코인
- 공모전 팀원 지원: +10 코인

## 7. 서비스 대상 / 타겟

- **주 타겟**: 경인지역(경기·인천) 대학생
- **주요 대학**: 아주대학교, 인하대학교, 경인교대, 가천대학교 등
